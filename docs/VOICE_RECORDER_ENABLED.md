## ⚠️ 常见问题解决

### 问题1：无法访问麦克风
**可能原因**：
- 浏览器未授权麦克风权限
- 麦克风被其他应用占用
- 系统设置禁用了麦克风

**解决方案**：
1. 检查浏览器地址栏，点击麦克风图标
2. 选择"允许"
3. 刷新页面重新请求权限
4. 检查系统设置 → 隐私 → 麦克风权限

### 问题2：录音功能不可用
**可能原因**：
- 浏览器不支持MediaRecorder API
- 使用了HTTP而非HTTPS
- 浏览器版本过旧

**解决方案**：
1. 确保使用HTTPS访问
2. 更新到最新版Chrome/Edge
3. 尝试其他浏览器（Chrome推荐）

### 问题3：录音质量差
**建议**：
- 在安静环境中录音
- 距离麦克风10-30cm
- 说话清晰，音量适中
- 避免背景噪音

## 🎯 最佳录音实践

### 录音准备
1. **环境选择**：安静的房间
2. **设备位置**：麦克风正前方10-30cm
3. **音量控制**：正常说话音量
4. **时间控制**：3-5秒最佳

### 录音内容建议
```
推荐内容：
- "欢迎使用音色复刻服务"（中性）
- "你好，这是测试音色"（简洁）
- "今天天气很好"（自然）

避免内容：
- 过长的句子
- 复杂的专业术语
- 情绪化的表达
```
