## 📊 系统概览

### 核心理念

将**游戏化成就系统**与**知识图谱**深度融合，构建精确到知识点的学习成就体系，同时通过学生画像展示全面成长，打造个性化、游戏化的智能学习体验。

### 三层成就体系

```
Level 1: 知识点成就 📍
  ├─ 精确到每个知识点
  ├─ 5个等级：青铜→白银→黄金→钻石→王者
  └─ 25个数学 + 25个英语 + 25个物理 = 75个

Level 2: 科目成就 📚
  ├─ 单科综合性成就
  ├─ 学科大师（数英物各1个）
  └─ 总计3个

Level 3: 学习习惯成就 🎯
  ├─ 跨科目通用维度
  ├─ 连续打卡、早起学习等
  └─ 5个（可扩展）

总计：83个成就
```

## 🔧 技术架构

### 数据库设计

**表结构**:
```sql
achievements           -- 成就定义
user_achievements      -- 用户成就
student_profiles       -- 学生画像
study_records          -- 学习记录
learning_progress      -- 学习进度（已有）
```

**关键索引**:
- `user_id + achievement_id` (复合唯一索引)
- `user_id + knowledge_point_id` (学习进度)
- `created_at` (时间查询)

### 后端技术栈

- **框架**: Next.js API Routes
- **ORM**: Prisma
- **数据库**: PostgreSQL
- **缓存**: Redis（可选）

### 前端技术栈

- **平台**: 微信小程序
- **图表**: ECharts for 微信小程序
- **组件**: 自定义成就组件
