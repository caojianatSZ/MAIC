## 五、家长端通知系统 ✅

### 5.1 订阅消息功能

**通知API**: `POST /api/notify/achievement`

**通知内容**:
- 成就名称
- 孩子昵称
- 掌握度等级
- 连续学习天数
- 科目信息

### 5.2 家长设置页面

**页面路径**: `/pages/parent-settings/parent-settings`

**功能**:
- 👨‍👩‍👧‍👦 绑定家长微信账号
- 🔔 通知开关设置
  - 成就解锁通知
  - 每日打卡提醒
  - 学习周报
- 📱 通知样式预览
- 💡 使用说明

### 5.3 通知流程

```
孩子解锁成就
  ↓
成就引擎触发
  ↓
检查家长绑定状态
  ↓
发送订阅消息
  ↓
家长微信收到通知
```
