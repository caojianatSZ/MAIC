# 图标资源说明

## TabBar 图标 ✅
已配置完成，包含以下图标：
- `tab-home.png` / `tab-home-active.png` - 首页图标
- `tab-progress.png` / `tab-progress-active.png` - 进度图标
- `tab-profile.png` / `tab-profile-active.png` - 个人中心图标

**注意**: 进度图标目前临时复用了原有的结果图标。建议后续替换为更符合"学习进度"主题的图标。

## 需要补充的资源

### 1. 默认头像 (default-avatar.png)
- **用途**: 个人中心页面用户头像
- **建议尺寸**: 200x200px
- **建议样式**: 圆形或支持CSS裁剪的方形

### 2. 默认课程封面 (default-cover.png)
- **用途**: 课程列表、学习进度页面的课程封面
- **建议尺寸**: 16:9比例 (如: 640x360px)
- **建议样式**: 教育主题，包含书本、学习等元素

## 临时方案

在正式资源生成前，可以使用以下方式：
- 使用纯色背景 + emoji 的SVG图片
- 使用在线占位图服务（不推荐生产环境）
- 使用CSS渐变背景

## 设计规范

参考 `app.wxss` 中的设计系统：
- 主色调: #6366F1 (靛蓝)
- 强调色: #EC4899 (粉红)
- 成功色: #10B981 (翠绿)
- 背景: #F9FAFB (浅灰)

建议图标风格与整体设计保持一致。
