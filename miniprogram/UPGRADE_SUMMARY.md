## 🚀 如何使用

### 1. 在微信开发者工具中预览
```bash
# 打开微信开发者工具
# 导入项目: /Users/caojian/Projects/OpenMAIC/miniprogram
# 编译并预览
```

### 2. 查看新页面
- **首页**: 点击Tab"首页"
- **播放页面**: 从首页点击课程卡片跳转
  ```
  pages/player/player?shareToken=xxx
  ```
- **进度页面**: 点击Tab"进度"
- **个人中心**: 点击Tab"我的"

### 3. 测试流程
1. 首页 → 拍照或输入题目
2. 提交 → 生成讲解（模拟）
3. 查看学习进度 → 点击课程卡片
4. 播放课程 → 场景导航 → 完成学习
5. 查看成就解锁

- **xs**: 8rpx
- **sm**: 16rpx
- **md**: 24rpx
- **lg**: 32rpx
- **xl**: 48rpx

### 圆角规范
- **小**: 12rpx (标签、徽章)
- **中**: 16rpx (按钮)
- **大**: 20rpx (输入框)

### 字体大小
- **小**: 22-24rpx (辅助信息)
- **中**: 28rpx (正文)
- **大**: 32rpx (小标题)
