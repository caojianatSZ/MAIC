## ⚠️ 常见问题处理

### 问题1：request:fail url not in domain list
**原因**：域名未在微信小程序后台配置
**解决**：按照第一步配置服务器域名

### 问题2：网络请求失败
**原因**：API地址配置错误或服务器不可访问
**解决**：
```bash
# 检查服务器状态
curl https://corp0.hz-college.com/api/health

# 确认API地址配置
cat miniprogram/config/env.js | grep baseUrl
```

### 问题3：页面无法加载
**原因**：页面未在 app.json 中注册
**解决**：检查 app.json 的 pages 配置
